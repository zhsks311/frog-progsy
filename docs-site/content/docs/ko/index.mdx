FrogProgsy는 Claude Code 앞에서 로컬로 실행됩니다. Codex/ChatGPT, Anthropic, xAI, Kimi, OpenAI 호환 API, 로컬 서버를 연결한 뒤 Claude Code는 평소처럼 쓰면 됩니다.

## FrogProgsy가 하는 일

- Claude Code는 원래 Messages 방식 그대로 둡니다.
- 요청을 사용자가 고른 AI 서비스와 모델로 보냅니다.
- 연결한 모델을 기존 Claude Code 모델 선택기에 보여줍니다.
- 요청 단계와 로컬 사용량을 안전하게 기록합니다.
- 선택한 모델이 웹 검색이나 이미지 이해를 못 할 때, 필요한 부분만 다른 모델이 대신 처리할 수 있습니다.
