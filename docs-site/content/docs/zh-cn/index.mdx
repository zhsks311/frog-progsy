FrogProgsy 在本机运行在 Claude Code 前面。连接 Codex/ChatGPT、Anthropic、xAI、Kimi、OpenAI-compatible API 或本地服务器后，继续照常使用 Claude Code。

## FrogProgsy does

- 保持 Claude Code 原生 Messages 接口不变。
- 将请求发送到你选择的 provider 和 model。
- 在 Claude Code 现有 model picker 中显示 routed models。
- 记录安全的本地 request activity 与 usage summary。
- 当所选模型无法直接处理 web search 或 image description 时，可使用显式 helper。
