FrogProgsy runs locally in front of Claude Code. Connect Codex/ChatGPT, Anthropic, xAI, Kimi, OpenAI-compatible APIs, or local servers, then keep using Claude Code as usual.

## Start here

- [Install frogp](/frog-progsy/getting-started/installation/)
- [Launch and verify](/frog-progsy/getting-started/quickstart/)
- [Connect an upstream](/frog-progsy/guides/providers/)
- [Choose route rules](/frog-progsy/guides/model-routing/)

## What FrogProgsy does

- Keeps Claude Code on its native Messages interface.
- Routes requests to the provider and model you choose.
- Shows routed models in the existing Claude Code model picker.
- Tracks safe local request activity and usage summaries.
- Supports opt-in helpers for web search and image descriptions when the selected model cannot handle them directly.
